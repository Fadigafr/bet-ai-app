# Guide de mise en ligne

## Option recommandée : Streamlit Community Cloud

1. Crée un dépôt GitHub.
2. Envoie tous les fichiers de ce dossier vers GitHub.
3. Va sur `https://share.streamlit.io/`.
4. Connecte ton compte GitHub.
5. Clique sur **Create app**.
6. Choisis ton dépôt, la branche `main`, puis le fichier `streamlit_app.py`.
7. Déploie l'application.

## Après déploiement
Tu obtiendras une URL web publique que tu pourras ouvrir depuis ton téléphone et partager à d'autres personnes.
