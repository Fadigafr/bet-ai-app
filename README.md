# Demo Sport Analytics Online

Prototype web **prêt à déployer** sur Streamlit Community Cloud.

## Fichiers
- `streamlit_app.py` : point d'entrée Streamlit
- `sample_matches.csv` : données d'exemple
- `requirements.txt` : dépendances
- `.streamlit/config.toml` : configuration thème
- `README_DEPLOY.md` : guide de mise en ligne

## Test local
```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```
